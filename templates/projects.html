At Carfagno Enterprises, I'm currently working on the development of a large language model (LLM)-powered trading platform designed to serve both beginner and expert-level traders. My primary focus has been on frontend development and UI/UX design, where I:

  • Build responsive, intuitive user interfaces using modern frontend frameworks (e.g., React, Tailwind CSS,) to ensure an engaging experience for users with varying levels of trading knowledge.
  • Collaborate closely with the design team to translate wireframes and mockups into clean, functional interfaces that prioritize usability and accessibility.
  • Integrate frontend components with backend APIs, ensuring smooth data flow between the LLM services and the user-facing platform.
  • Implement interactive elements and data visualizations (e.g., charts, trade insights) to make complex information digestible for users.
  • Continuously refine UI/UX based on user feedback and testing, balancing aesthetics with performance and functionality.
